 ## Installation
 1. Install [Node](https://nodejs.org/en/) 
 2. Clone this repository `git clone https://github.com/grability-inc/a2censo-automation.git` 
 3. Install dependencies  `npm i`
 
## To test
```sh 
$ npm run test
```
